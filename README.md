# useful-prompts

A small collection of system prompts I use with capable models (Claude, Codex,
GPT-class) for software work. Paste one into a model at the start of a session
and let it drive.

## Prompts

### [roadmap_generator.md](roadmap_generator.md)
Turns the model into a **product roadmap partner**. It interviews you about
your app idea — concept, users, scope, tech stack, look-and-feel, accessibility
— then generates a persistent `ROADMAP.md` that you and the AI maintain
together across future sessions.

Use it when you're starting a new project and want a real plan instead of a
wall of bullet points.
